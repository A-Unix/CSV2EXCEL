## Installation instructions(For linux only)
 ### Clone the repo
  ```bash
  git clone https://github.com/A-Unix/CSV2EXCEL.git
  ```
 ### Move to cloned folder
 ```bash
 cd CSV2EXCEL
 ```
 ### Get permissions
  ```bash
  chmod +x main.sh
  ```
 ### Install dependencies
  ```bash
  ./main.sh
  ```
